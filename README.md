# the_book

## Prereq
```
# Creating virtual env called dev_env
$ python -m venv dev_env
# Start virtual env
$ source dev_env/bin/activate
# Deactivate virtual env
$ deactivate
```


## installing dependencies
```
$ ./setup.sh
```
** only run this when you create and start virtual env **
